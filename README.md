# Terminal_operating_system

## Description

**Terminal_operating_system** est un projet développé en Python qui simule un petit système d'exploitation fonctionnant dans le terminal. Il permet d'exécuter différentes commandes et de découvrir les principes de base d'un système d'exploitation à travers une interface en ligne de commande.

## Fonctionnalités

* Interface en ligne de commande.
* Exécution de commandes personnalisées.
* Affichage de messages et d'informations système.
* Architecture simple et facile à comprendre.
* Projet conçu à des fins d'apprentissage.

## Technologies utilisées

* Python 3

## Installation

1. Téléchargez ou clonez ce dépôt.
2. Assurez-vous que Python 3 est installé sur votre ordinateur.
3. Ouvrez un terminal dans le dossier du projet.

## Exécution

Lancez le fichier principal du projet avec Python.

## Structure du projet

* `main.py` : point d'entrée du programme.
* Autres fichiers : contiennent les différentes fonctionnalités et commandes du système.

## Objectif

Ce projet a été réalisé afin d'améliorer les compétences en programmation Python, en conception de logiciels et en organisation du code.
